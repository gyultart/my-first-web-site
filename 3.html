    Tag란

    웹을 지배하는 가장 중요한 문법 <열리는 태그><닫히는 태그>
    1. 볼드체: 볼드처리
    2. 밑줄: 언더라인
    3. 제목: h1~h6, 숫자가 커질수록 글씨가 작아지는 제목 태그
    4. 단순한 줄바꿈: html에서 줄바꿈을 하기 위해서 br tag를 씀, 닫힘 태그 필요 없음
    5. Paragraph Tag: 어디부터 어디까지가 한 단락인지 알려줌. 정보로 단락을 묶어주는 것. 정해진 여백만큼, 시각적으론 자유도가 떨어지지만, CSS를 사용하면 됨.
    6. image:

    - br tag 줄바꿈 TEST: 원하는만큼 br을 붙여주면 됨

    - p tag 줄바꿈 TEST: 원하는만큼 css tag사용

    html이 중요한 이유:css코드로 디자인만 된 정보 VS html tag로 중요도가 표시된 정보가 검색 엔진에 놓였을 때, 중요한 정보라고 판단되어 상위에 걸리게 됨. 웹은 시각장애인이 사람들에게도 정확한 정보를 Tag를 활용해 지정해줘야 정보 전달이 가능하다.
